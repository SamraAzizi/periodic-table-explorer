## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/periodic-table-explorer.git
cd periodic-table-explorer

```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open in your browser:
```bash
http://localhost:3000
```
